# Exercícios de Java - Meu Aprendizado

Este repositório contém uma coleção de exercícios em Java que foram desenvolvidos durante o meu processo de aprendizado da linguagem. Cada exercício foi criado para praticar e consolidar conceitos importantes de programação orientada a objetos (POO) e outras funcionalidades da linguagem Java.

## Conceitos Abordados

Nos exercícios presentes neste repositório, foram aplicados os seguintes conceitos:

- **Herança**: Criação de classes que herdam atributos e métodos de outras classes.
- **Polimorfismo**: Uso de métodos com o mesmo nome, mas com comportamentos diferentes em classes derivadas.
- **Tratamento de Exceções**: Uso de `try-catch` para lidar com erros e exceções.
- **Encapsulamento**: Proteção de dados através do uso de modificadores de acesso (`private`, `public`, `protected`).
- **Classes e Objetos**: Definição de classes, criação de objetos e uso de métodos e atributos.
- **Collections**: Uso de estruturas de dados como `ArrayList`, `HashMap`, etc.
- **Interfaces e Classes Abstratas**: Implementação de interfaces e uso de classes abstratas.
